Características Principales

Uso de Guard en la App

Implementación de AuthGuard para proteger rutas y simular autenticación segura.

Uso de Local Storage

Persistencia de datos del usuario y estado de la aplicación usando localStorage.

Integración con Backend

Conexión con servidores Go mediante servicios HTTP y endpoints RESTful.

Uso de Señales de Angular

Manejo reactivo de estado con signals para componentes dinámicos y observables RxJS.
